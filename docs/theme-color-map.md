# Omarchy Color Map (Flat OneDark)

## Scope
This document maps where Omarchy reads colors from at runtime and how this repository generates those files.

## Theme Flow
1. Edit `theme.tokens.sh`.
2. Run `./scripts/build-theme.sh`.
3. Apply with `./scripts/apply-theme.sh`.
4. The apply script syncs the repo to `~/.config/omarchy/themes/flat-onedark`, updates the live Waybar/Starship configs with backups, then calls `omarchy-theme-set flat-onedark`.
5. Omarchy copies the theme to `~/.config/omarchy/current/theme`, generates template files from `colors.toml`, and restarts affected components.

## Source Types
- Direct outputs from this repo: generated directly by `scripts/build-theme.sh`.
- Template outputs: generated by `omarchy-theme-set-templates` using `colors.toml` and Omarchy templates (`~/.local/share/omarchy/default/themed/*.tpl`).

## Waybar Policy
- Base mode:
  - `waybar-theme/config.jsonc` mirrors the current Omarchy stock layout semantics without forcing qbar.
  - `waybar-theme/style.css` is the stock-safe wrapper that imports `waybar.css`.
- qbar policy:
  - qbar is no longer installed or configured by this theme.
  - Use `qbar setup` to let qbar own its icons, modules and Waybar integration.
  - `qbar.css` is generated as a future-facing contract for qbar to consume later, with fallback to qbar's current self-managed styling.

Reset to default Waybar mode:

```bash
omarchy-refresh-waybar
```

## Hardcoded Color Policy
- `theme.tokens.sh` is the only place to edit palette values.
- `scripts/build-theme.sh` must generate all repo files that encode colors.
- The live Waybar layout file is generated as `waybar-theme/config.jsonc`.
- qbar's visual contract is generated separately as `qbar.css`, without changing the live Waybar layout.

## Useful Commands
```bash
omarchy-theme-current
omarchy-theme-list
./scripts/apply-theme.sh
./scripts/apply-theme.sh --dry-run
qbar setup
omarchy-refresh-waybar
omarchy-restart-waybar
omarchy-restart-walker
omarchy-restart-mako
omarchy-restart-swayosd
omarchy-restart-terminal

```
